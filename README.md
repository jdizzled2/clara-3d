# How to Install

1. Install node.js from "https://nodejs.org/en/"
2. Install VS Code
3. Run `npm i -g pnpm` this installs all the packages that you guys need
4. Run `pnpm i`

# How to Run

1. Run `pnpm dev`
2. Open `http://localhost:3000`
